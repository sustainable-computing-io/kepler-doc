    v0.10.0+ 版本中的变化

    Kepler 在 v0.10.0 版本中进行了重大架构重写，引入了：

    • 面向服务的设计 替代了以 eBPF 为中心的架构
    • 简化的配置 使用 CLI 标志 + YAML 而非 40+ 个环境变量
    • 精简的指标 采用专注于 CPU 的归因方式，而非复杂的组件分解
    • 新的开发工作流 使用 Docker Compose 环境

    使用此文档
