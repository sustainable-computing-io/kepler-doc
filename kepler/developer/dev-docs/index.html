    üõ†Ô∏è Quick Developer Setup

    For immediate development setup, the key commands are:

    # Clone and build
git clone https://github.com/sustainable-computing-io/kepler.git
cd kepler
make build

# Development with Docker Compose (includes Grafana + Prometheus)
cd compose/dev
docker compose up -d

# Run tests
make test

# Local development
sudo ./bin/kepler --config.file hack/config.yaml
