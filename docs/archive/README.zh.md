## v0.10.0 版本的重大变更

Kepler 在 v0.10.0 版本进行了架构重构，主要变更包括：

- **服务导向设计** 取代了以 eBPF 为中心的架构
- **简化的配置** 采用 CLI 参数 + YAML 替代 40 多个环境变量
- **精简的指标** 聚焦 CPU 能耗归因而非复杂的组件细分
- **新的开发流程** 采用 Docker Compose 环境
